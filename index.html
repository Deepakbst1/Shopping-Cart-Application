Meshop is a project to check anything that require while developent and developing a E-Commercial Website , data will be with the help of JSON and redirect to portal , the dummy data will be available and the project repository into github, Signup and login handle using local storage - multiple users - if not logged in donâ€™t let the person enter the shop or my cart or profile, Filter available based on color,size,price.

Main Backbone of any E-Commercial platform is its storage, as it is required to store the user details for validation and authetications purpose, product details and many more. The local storage of the browser is used to store the user details such as email and password, current user details, product data that is fetched from the server and the cart details of the products which is added by user choice.

Features

  • Signup and Login level authentications such as the user with two same email id cannot signup, email and password check while login and other basic authetications are also provided. If in any case, the user deletes the local storage data, the website will automatically redirect the user to the landing page on refreshing. Same user can signup and login with multiple emails and password. Every credentials will be stored in the local storage from which the authentication will be done.
  • The shopping portal can only be accessed if the user is logged in, else the user will be redirected to the landing page.The shopping portal has the features such as searching the product, filtering and categorizing the products based on given filter and category tags.
  • There will be a functionality cart icon will we available only user add somthing to the cart.Every product in portal has Add to Cart button, where the user can add products in multiple quantity and remove it from the cart as well.
  • We can initiate the dummy payment procedure using the RazorPay API in test mode from the cart page only.
  • Designing of the website is totally depend on the basic front-end technologies used as HTML,css, and Javascript.
